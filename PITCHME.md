## History

参考 : [Ruby(Programming Language) - wikipedia](https://en.wikipedia.org/wiki/Ruby_(programming_language))

* 1993/02 : Ruby 作り始める

* 1996/12 : Ruby 1.0
* 1998/12 : Ruby 1.2
* 1999/08 : Ruby 1.4
    * ピッケル本発売

* 2013/02 : Ruby 2.0
